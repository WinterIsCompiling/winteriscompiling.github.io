About TaddyPole

TaddyPole is an educational game, created for Tadpoles.org.uk to help teach children about the lifecycle and habitats of tadpoles.

TaddyPole was created using the Unity game engine, and is available to play in any modern web browser. TaddyPole can also be downloaded on iOS, Android, and Windows 10, through the store links above.
