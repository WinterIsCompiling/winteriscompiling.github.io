About TaddyPole

TaddyPole is an educational game, created to help teach children about the lifecycle and habitats of tadpoles.

TaddyPole was created using the Unity game engine, and is available to play in any modern web browser. TaddyPole is also available as a direct download for Android, and is coming soon for iOS.
